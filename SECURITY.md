# Security Policy

## Scope

`denex-network-manager` is a local development tool. It manages Docker containers on your local
machine and handles Keycloak admin credentials and Docker socket access. It is not intended for
production deployments.

Default credentials in configuration examples (e.g. `admin`/`admin`) are for local development only
and must not be used in any internet-accessible environment.

## Reporting a Vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report security issues via
[GitHub Security Advisories](https://github.com/denex-io/denex-localnet/security/advisories/new).

We will acknowledge receipt within 48 hours and aim to provide a fix or mitigation within 14 days
for confirmed vulnerabilities.
